# Omer Zilbershtein — AI Research Scientist Portfolio @ Pixellot

A targeted one-pager applying for the **AI Research Scientist (Student)** role at Pixellot.

## Run Locally
```bash
npm install
cp .env.example .env.local   # edit with your backend URL
npm run dev -- --port 3002
```

## Tech Stack
Next.js 16 · Tailwind CSS 4 · Framer Motion · Aceternity UI · FastAPI + LangGraph backend
